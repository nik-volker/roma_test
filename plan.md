## 📋 Обновленный план: AI-ассистент для психолога (MVP)

### **TL;DR**
Простой веб-чат (HTML/CSS/JS на GitHub Pages) → Flask API → OpenAI API (GPT-4.1 mini). Пользователь описывает проблему, AI определяет состояние, задает вопросы, предлагает техники. Контекст только в браузере (localStorage). Развернуться и тестировать за неделю.

---

### **1️⃣ Обязательные функции MVP**

**Core:**
- Chat UI (отправка/получение сообщений)
- Классификация состояния (8 типов из твоего списка)
- 2-4 уточняющих вопроса
- Рекомендация конкретной техники / упражнения
- Поддерживающий диалог

**Safety:**
- Детекция кризиса (суицид, self-harm)
- Кризисное сообщение с общими рекомендациями (без телефонов)
- Логирование risk-сигналов

---

### **2️⃣ Архитектура (простая)**

```
Браузер (localStorage: история чата)
    ↓ POST /api/chat (message + контекст)
Flask API (Python 3.9+)
    ↓ вызов OpenAI API (GPT-4.1 mini)
OpenAI (GPT-4.1 mini)
    ↓ JSON ответ
Flask возвращает JSON: 
{
  "message": "...",
  "detected_state": "anxiety_in_relationship",
  "suggested_technique": "Box breathing",
  "risk_level": "none"
}
```

**Хранение**: контекст полностью в памяти браузера + текущий разговор отправляется серверу. Между сеансами ничего не сохраняется.

---

### **3️⃣ User Flow**

1. Открыл https://[github]github.io/ai-psycho/
2. Написал про ситуацию ("Мы не говорим друг с другом 3 дня")
3. AI ответил эмпатично + 2-4 уточняющих вопроса
4. Ответил на вопросы
5. AI: определил состояние ("напряжение после конфликта") + техника ("безопасный разговор")
6. Может таскать дальше, закрыть → чат удалится

**Если кризис** → специальное сообщение + общие рекомендации вместо обычного ответа

### **5️⃣ Промпты**

**System prompt** (in `prompts.py`): Четко описываешь роль AI, уточняешь что НЕЛЬЗЯ делать (диагнозы, манипуляция, сторонничество), объясняешь 8 состояний, требуешь JSON ответ с `detected_state`, `suggested_technique`, `risk_level`.

**Техники по состояниям**:
```python
TECHNIQUES = {
    "anxiety_in_relationship": "Box breathing (вдох 4, задержка 4, выдох 4, пауза 4)",
    "resentment_after_conflict": "Переформулирование перспективы (спроси себя...)",
    "distance_coldness": "Техника микромоментов близости (найди маленькое действие...)",
    # ... и т.д. для всех 8
}
```

### **6️⃣ Safety-логика**

**Детекция кризиса** (в `safety_check.py`):
- **HIGH**: "убить себя", "покончить с собой", "давай мне яд"
- **MEDIUM**: "не хочу жить", "не вижу смысла"
- **NONE/LOW**: нормальные эмоции

**Если HIGH или MEDIUM**:
```
🛡️ Мне важно, что ты мне рассказал. 

Я не могу помочь в кризисе. Настоятельно рекомендую обратиться к психологу, психиатру или в полицию.
```

Всё залогируется, но пользователь не узнает, что его классифицировали как risk.

### **8️⃣ Ключевые решения**

- **Flask** — простой, хороший OpenAI SDK для Python
- **No database** — MVP не нужна история между сеансами
- **localStorage** — приватность + простота
- **Plain JS** — неделя, минимум зависимостей
- **GitHub Pages + Render** — фронт статичный, бэк отдельно (GitHub Pages не поддерживает бэкенд)
